Biophysical Essentials (BPE)

What is Biophysical Essentials?

Biophysical Essentials is an open-source software written in Python QT. It enables researchers to analyze biophysical data, such as Patch-Clamp recordings from amplifiers made by different companies, including HEKA, Sutter, Axon, and more.

How to Install Biophysical Essentials

Biophysical Essentials is currently available for Windows, Linux, and MacOS. To install it, follow the steps describe below:

  1. Clone the repository from https://github.com/ZiDa20/Biophysical_Essentials
  1. Install the specific environment.yml (Linux, MacOS or Windows) file for your operating system using the command conda create env -f enviroment.yml.
    1. This should install Python 3.11
    1. PySide6, Pythons QT-Package
    1. DuckDB 0.7.1 (https://duckdb.org) OLAP Local Database Manager
    1. All necessary dependencies
  1. After the conda enviroment is created successfully navigate to ./src and start the tool using the following command: python start.py
  1. For Developers: If you want to contribute to the Project please feel free to open pull requests. Please always ensure that Tests are running properly by starting the Unit Tests using python -m unittest discover ../Tests/frontend_page_tests.py

Important to Note: BPE will be available as on-click installer for Windows and MacOS later this year!

Biophysical Essentials Overview

Once installed, the software can be used to analyze biophysical data from Patch-Clamp recordings using amplifiers from different companies. The landing page of the software can be viewed in either White Mode or Dark Mode.
